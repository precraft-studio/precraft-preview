3. **Stats** — 25+ / 500+ / 001554 三大數據
4. **Services** — 工商登記 + 帳務處理 兩大類，含 editorial serif 引文
5. **Why Us** — 4 張理由卡片（合法/經驗/客戶/收送）
6. **CTA** — 深色背景 + 聯絡資訊 + 主按鈕
7. **Footer** — 4 欄資訊架構

## 設計原則

- **白為主，深靛藍點綴，深翠綠強調，暖金極少**
- 不使用紅色（不適合財稅業）
- 不使用花俏漸層或過度陰影
- 字型粗細控制在 300-600，避免超粗
- editorial 局部字型只在「服務細節說明」出現
